JREインストールガイド

　JREとはJAVAのプログラムを実行するためのランタイムであり
　JAVAのプログラムを実行するPCには必ずインストールが必要です。
　JREの入手は下記サイトから行います。

　　　http://www.java.com/ja/download/

　サイト内のをクリックしてダウンロード。

　ダウンロードしたファイルを実行し、JREをインストールします。
　インストール時の選択は全てデフォルト設定でかまいません。
